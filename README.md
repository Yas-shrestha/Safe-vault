# Secure Password Manager

A simple and secure password management application that allows users to **store, manage, and generate strong passwords**. The application focuses on protecting user data using modern encryption and hashing techniques.

User authentication is protected with **bcrypt hashing**, while stored passwords are encrypted using **AES encryption** to ensure sensitive information remains secure.

---

# Features

## User Authentication

- Secure user login and registration
- Passwords are hashed using **bcrypt**
- Protects against password leaks and database compromise

---

## Secure Password Storage

Users can safely store credentials such as:

- Website name
- Username or email
- Password
- Notes (optional)

Stored passwords are encrypted using **AES encryption**, ensuring that sensitive data cannot be read directly from the database.

---

## Password Generator

The application includes a **built-in password generator** that can create strong and random passwords.

Features include:

- Adjustable password length
- Random character generation
- Strong security patterns
